## UnoZero Design Notes
This section documents a few recurring hardware design choices shared across the Functional Boards, for anyone interested in the reasoning behind specific component values.

### Input Biasing (Series/Pull-up Resistors)
Boards that generate a logic line from a mechanical switch (**SF-02**, **SF-03**, **SF-06**, **SF-09**) use a resistor pair sized directly from the TTL-LS datasheet thresholds ($V_{IH} \geq 2V$, $V_{IL} \leq 0.8V$,
$I_{IH} \leq 20\mu A$, $I_{IL} \leq -0.4mA$):
- **Pull-up ($47k\Omega$):** keeps the line comfortably above $V_{IH}$ when the switch is open, with margin for input leakage current and noise immunity. The selected value is well below the maximum pull-up resistance derived from the TTL-LS input leakage specifications.
- **Series resistor ($100\Omega$):** the current sourced by the TTL input when driven low is fixed by the TTL input gate ($I_{IL}$). The resistor is sized small enough that this current produces a negligible voltage drop, keeping the line comfortably below $V_{IL}$.

### LED Indicators
LED current-limiting resistors (**SF-01**, **SF-04**, **SF-07**) are sized to stay within the TTL-LS output drive limits ($I_{OL} \leq 8mA$ sinking, $I_{OH} \leq -0.4mA$ sourcing). For example, on **SF-04** a $1k\Omega$ resistor yields roughly 2.5mA when the gate output sinks current (LED ON), a safe margin below $I_{OL}$, chosen for adequate brightness while remaining comfortably within the guaranteed TTL-LS sink current capability.

### Transistor Stage (SF-09)
Q1 (small-signal NPN transistor) works as a simple inverting buffer: it converts the external **active-low** Clock/Reset control line into the **active-high** level required by the 74LS90's reset inputs (R01/R02). R1 biases the base from the incoming control line, R2 sets the high level at the collector when Q1 is
off, and R3 ($100\Omega$) protects the IC input, following the same sizing logic described above.

### Debounce & Monostable Network (SF-06)
The Pulse Generator combines two stages built from NOR gates:
- An RC input filter ($100\Omega$ + $0.1\mu F$) attenuates the fast transitions generated by mechanical contact bounce before the signal reaches the logic gates.
- A cross-coupled NOR/RC monostable stage ($22k\Omega$ + $1\mu F$) shapes the debounced edge into the fixed ~15ms output pulse described in the SF-06 README.

### Notes on Notes
These values are not unique: several nearby standard values would also operate correctly. The selected values were derived from worst-case TTL-LS datasheet parameters to provide comfortable operating margins while keeping the circuits simple and suitable for home construction.
